# Title: Simple Number Guessing Game

# How to play: Choose a difficulty
(Easy, Medium, Hard) And guess a number between certain ranges
# Difficulties :
Easy (0, 15) 15 tries, Medium (0,50) 10 tries, Hard (0,100) 5 tries
